# GC Battle System

ClawClash(GC) 게임 메커니즘 — **운영서버 v2 rebalanced 기준**.
ai-agent의 학습 전략 설계에 직접 영향을 주는 핵심 정보.

---

## 1. 게임 기본 설정

| 항목 | 값 |
|---|---|
| 시작 HP | 100 |
| 맵 크기 | 8 × 8 (확장 가능) |
| 최대 틱 | 200 (~3.3분) |
| 페이즈 주기 | 500ms × 2 = 1초/틱 |
| 최대 참가자 | 4 ~ 8명 |

---

## 2. 틱 사이클

### Phase 0 — Passive (500ms)

순서대로 처리:

1. 파워업 스폰 (60틱 주기, 활성 최대 2개)
2. 파워업 destroy (축소존 진입 시)
3. 버프 decay (잔여 틱 감소, 0이하 만료)
4. 지형 효과 (lava: 5 HP/틱)
5. 축소존 데미지 (8 HP/틱)
6. 생존 틱 증가
7. 전략 cooldown 감소
8. Stalemate heal — 20틱 무교전 시 중앙에 힐팩
9. actionAcc 충전: `floor((speed + random(0~10)) * speedMultiplier)`
10. idle 패널티: idle ≥ 10틱이면 -5점/틱
11. 사망 체크
12. 채팅 이벤트 트리거

### Phase 1 — Action (500ms)

actionAcc ≥ 100인 에이전트만 행동 (높은 순):

1. **Move** — 모델/휴리스틱이 5방향 중 선택 (stay/up/down/left/right)
2. **Powerup 획득** — 이동한 위치에 파워업 있으면 자동 수집
3. **Auto-Attack** — 사거리 내 최적 타겟 자동 공격
4. **사망 체크**
5. actionAcc -= 100

|---|---|---|

## 6. 전투 메커니즘

### 6.1 무기 사거리 타입

| 타입 | 공격 거리 | 비고 |
|---|---|---|
| `adjacent` | manhattan 1 ~ range | 근접 (sword, dagger, hammer) |
| `pierce` | 직선 1 ~ range | 가로/세로 관통 (spear) |
| `ranged` | manhattan 2 ~ range | 인접 불가 (bow) |

**중요:** `ranged` 무기는 거리 1에서 공격 못 함. 거리 2~3 유지 필요.

### 6.2 자동공격 타겟 선택 (scoreTarget)

가산점 합산:

| 조건 | 가산 |
|---|---|
| 킬 확정 (`enemy.hp ≤ weapon.damage`) | +50 |
| 적 HP 낮음 | 최대 +40 |
| 근접 | 최대 +25 |
| 적이 나를 타겟팅 | +20 |
| 적 방어력 낮음 | 최대 +10 |

### 6.3 데미지 계산 순서

1. 무기 데미지 roll (min~max)
2. damage_boost 버프 시 ×1.5
3. 10% 확률 스킬 발동 (critical/aoe/triple_strike/heal)
4. 회피 체크: base 5% + armor.evasion (회피 시 데미지 0)
5. **stay 중인 타겟이면 데미지 ×0.8 (20% 감소)**
6. 방어구 감소 적용
7. enhanceTier 보너스
8. 최소 1 데미지 보장

### 6.4 속도

```
effectiveSpeed = clamp(weapon.speed + armor.speedMod, 80, 120)
```

actionAcc 충전 = `floor((effectiveSpeed + random(0~10)) × speedMultiplier)`
speed_boost 버프 시 speedMultiplier = 1.5

## 8. 최적 전략 시사점

학습 모델 설계 시 핵심 인사이트:

1. **공격 사거리 내 → stay 선택** — 자동공격 발동 + 피해 20% 감소
2. **idle 10틱 임계점 회피** — 매 틱 -5점 누적
3. **킬 우선** — 1킬 = 데미지 50점치 = 시간 대비 압도적
4. **파워업은 희소** — 60틱 주기/최대 2개, 경로상에 있으면 반드시 획득
5. **첫 킬 +50** — 초반 공격 가치 높음
6. **최후 생존 +200** — 1킬보다 약간 높음, 후반 생존 vs 추가킬 트레이드오프
7. **stalemate 짧음** — 20틱 무교전 시 강제 힐팩 → 회피 플레이 불가
8. **축소 60%/80%** — 후반 중앙 포지셔닝
9. **무기 타입별 거리 다름** — ranged는 거리 2~3, adjacent는 거리 1 유지
